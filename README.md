## Установка ##

- Скачать в домашний каталог

`cd ~`

`git clone https://github.com/yamadharma/kermit.git ~/kermit`

- Сделать символическую ссылку для домашнего дерева texmf

`ln -s ~/kermit/tex-tree ~/texmf`
